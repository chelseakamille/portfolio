Development

When I finished creating and designing the wireframes on Adobe XD, I was thrilled to start coding and bring the design to life. For majority of the layout, I chose to use flexbox because it suited my design better. Using flex direction made it easier for me to align the contents in the center and helped with making the entire page responsive.

For the images and large headings, I decided to use vw instead of a fixed width for a more seamless experience when resizing the pages. Using this method saved me a lot of time as I didn't have to target all of these elements again when I was doing the media queries for smaller screens. I only had to adjust certain images and headings that I wanted bigger on mobile screens.
